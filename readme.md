# AdminPro - Notas informativa

Correr el programa (Recordar ejecutar) ````npm install````

## Install
````npm install express````

## nodemon
````npm install -g nodemon````

## Correr app
````npm run start:dev````

## Conectar mi BD con mi servidor
````npm i mongoose@5.9.21 --save````

## Crear y usar .env
````npm i dotenv@8.2.0````

## CORS
````npm i cors@2.8.5````

## Verificar user
Primero loguerse para obtener el token
